# SeedOS

A terminal-based puzzle exploratation game. Traverse a Linux style shell to vanquish aa mysterious virus that's infected your system!

# Install

```bash
git clone https://github.com/Al0102/TermProject.git
```

# Play

The terminal should be maximised to 100 columns and 30 rows for the best experience
Please have sound ON.

Install Python 3.12+ if you don't have it already.

Clone the repo.
`git clone https://github.com/Al0102/SeedOS-Game.git`

Navigate to it.
`cd SeedOS-Game`

Install dependencies
```bash
# Optionally activate a virtual environement
python3 -m venv .venv
source .venv/Scripts/activate.sh
pip install
python3 -m game.game
```

## Controls

The game uses Arrow Keys for movement and Return is to confirm/submit/select.
For text input, your cursor position is indicated by an underscore, "_"
